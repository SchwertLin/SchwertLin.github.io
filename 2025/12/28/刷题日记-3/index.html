Q3-使数组和能被p整除

link

思路：我得到前缀和数组 s，遍历 curr，找一个最靠近 curr 的 prev，使得 (s[curr] - s[prev]) % p == x， 也就是 s[prev] % p == (s[curr] - x + p) % p，用 curr - prev 更新最小长度。
